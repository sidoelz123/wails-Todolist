## üìÅ Getting Started

### Installation

1. Clone the repo
   ```sh
   https://github.com/sidoelz123/wails-Todolist.git
   ```
2. Navigate to the Project Directory

    ```sh
    cd wails-Todolist
    ```

3. Install Depedencies
   ```sh
   go mod tidy
   ```

4. Start the project
   ```sh
   wails dev
   ```
